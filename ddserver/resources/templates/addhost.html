Set a password

Each hostname gets its individual password used to update its IPv4 address. The minimum password length is {{config.auth.password_min_chars}} characters. If you want to update multiple hostnames at once, you have to use the same password for each of them.

(Optional) Set an initial IPv4 address

If you specify an initial IPv4 address, your hostname will instantly be available by this address. If you leave the IPv4 address field blank, you will have to send an update using your homerouter or the ddclient program, before the hostname can be resolved using to DNS queries.

(Optional) Set a static IPv6 address

You can provide a static IPv6 address for your hostname. The IPv6 address can only be set using the Web-UI. Dynamic updates are not possible at the moment.

(Optional) Add a description

A description can be set to add personal notes. The description is only visible in the ddserver Web-UI.

You are done

As soon as the hostname has been added, you can modify it or display configuration examples by navigating to the Your hostnames tab and click the Details button for your hostname.
